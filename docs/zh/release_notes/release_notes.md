## 特性变更说明

### 26.0.0

无新增特性。

### 8.3.0

首次发布，功能点如下：

1. 提供核心追踪API，支持代码执行点标记和范围测量。

2. 提供域(Domain)管理API，对标记和范围进行分类，用于在多工具场景下隔离不同数据类型。

3. 提供内存追踪API，支持内存区域的注册和注销，用于内存调试工具。
